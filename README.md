# Electricity Bill Analyzer

## 📌 Project Overview
The **Electricity Bill Analyzer** is a C++ console-based application designed to manage customer electricity records and automate bill calculation. It helps in tracking electricity consumption, generating bills, and maintaining payment status for multiple customers in a structured way.

This project is built using **Object-Oriented Programming (OOP)** concepts and is suitable for understanding real-world system design in C++.

---

## 🚀 Features

### 👤 Customer Management
- Add new customer records
- Update existing customer details
- Delete customer from system
- View individual customer details

### ⚡ Billing System
- Store previous and current meter readings
- Automatically calculate electricity units consumed
- Generate total bill based on units consumed
- Display detailed bill/receipt

### 📊 System Reports
- Display all customers in the system
- Show paid customers
- Show pending customers
- Count total customers, paid, and unpaid users

### 💾 Data Handling
- Uses C++ classes and vectors for structured data storage
- Modular functions for clean and maintainable code

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
  - Classes & Objects
  - Encapsulation
- Vectors (STL)
- Functions & Modular Programming
- Conditional Logic & Loops
- Menu-driven program structure

## 🖥️ How to Run the Project

### 🔧 Using g++ compiler:

1. Clone the repository:
```bash
git clone https://github.com/Savitasaini02/Electricity-Bill-Analyzer.git
```

2. Navigate to the projects folder :
 ```bash
   cd Electricity-Bill-Analyzer
   ```

3. Compile the code:
```bash
g++ ElectricityBillAnalyzer.cpp -o bill
```

4. Run the program:
```bash
./bill
```
